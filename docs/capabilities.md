## Extension seams

To add a connector, place it at `connectors/{category}/{name}.py` and implement the `BaseConnector` Protocol **structurally** (a bare class ‚Äî do NOT subclass it).

### Protocol seams

#### `BaseConnector` (machina.connectors.base)

Protocol that all Machina connectors must satisfy.

- `capabilities(...)` ‚Äî Typed set of actions this connector supports.
- `async connect(...)` ‚Äî Establish a connection to the external system.
- `async disconnect(...)` ‚Äî Gracefully close the connection.
- `async health_check(...)` ‚Äî Check whether the external system is reachable and responsive.

#### `RefreshableConnector` (machina.connectors.docs.watcher)

Any connector with a synchronous refresh method.

- `refresh(...)`

#### `SupportsConfirmation` (machina.connectors.comms.types)

Channels that can confirm a pending write synchronously, in-turn.

- `async request_confirmation(...)` ‚Äî Ask the user to confirm a pending write and return their decision.

### Convention seams

Not reflectable Protocols ‚Äî conventions an implementer follows, with where each one lives:

- **MCP tool registration** ‚Äî Map a capability to the MCP tool(s) auto-registered when a connector declares it.
  - location: `mcp/tools.py::CAPABILITY_TO_TOOL`
- **capability vocabulary** ‚Äî Add a new action identifier to the Capability StrEnum.
  - location: `connectors/capabilities.py::Capability.{NEW_MEMBER}`
- **capability‚Üímethod map** ‚Äî Map the new capability to its backing method name (capability values are not always method names).
  - location: `introspect/_methods.py::CAPABILITY_TO_METHOD`
- **connector-type registration** ‚Äî Register a new connector type ‚Üí dotted factory path so the runtime and MCP layer can discover it.
  - location: `runtime._CONNECTOR_FACTORIES['{type}'] = '{dotted.path.ConnectorClass}'`
- **transport/mapper split** ‚Äî Vendor payload‚Üîdomain mapping lives as pure functions, separate from the connector's transport code; the connector imports the mapper.
  - location: `connectors/cmms/mappers/{vendor}.py`
- **workflow builtins** ‚Äî Drop a built-in workflow template into the builtins package.
  - location: `workflows/builtins/{workflow_name}.py`
